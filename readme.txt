`To run the code:`
1- Open the .ino file in the arduino IDE and setup the wifi network credentials in the 'ssid' & 'password' fields in lines 18 and 19
2- Inside the IDE, go to 'tools', and then press the 'ESP32 Sketch Data Upload' button and wait for the message confirming that the upload is complete
3- Upload the code to the ESP32 using the upload button and wait for completion
4- Type the IP address, broadcasted in the serial monitor, in the URL of a web browser to access the embedded website
